# HF-MCP9700 documentation

This driver implements the standard MCP9700/MCP9700A transfer function from the Microchip datasheet:

- Typical output at 0 °C: **0.5 V**
- Typical sensitivity: **10 mV/°C**

Your platform must measure **Vout in volts** at the MCP9700 output (after any external scaling) and pass that value through `ReadChannelV`.
