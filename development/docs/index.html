HF-MCP9700 documentation

Welcome. This site mirrors the docs/ folder in the repository and adds navigation, search, and diagrams for the MCP9700 / MCP9700A linear active thermistor driver.

Browse on GitHub: repository home · Issues

Documentation structure

Getting started

  1. Installation — Compiler/CMake requirements and include layout
  2. Quick start — Smallest adapter you need to read °C
  3. Transfer function — Equations, typical curve, calibration knobs

Hardware & integration

  1. Hardware setup — ESP32-C6 — Wiring diagram, ADC channel, attenuation
  2. CMake integration — hf_mcp9700_build_settings.cmake, hf::mcp9700 target

Reference & examples

  1. API reference — Mcp9700Thermistor, adapter contract, version helper
  2. Examples — ESP-IDF example and build/flash flow
  3. Troubleshooting — Version header, ADC range, ReadChannelV semantics

Manufacturer

  1. Datasheet & links — Official Microchip documentation and product pages

Recommended reading order

  1. Installation — ensure mcp9700_version.h generation is wired
  2. Hardware setup — connect Vout to a suitable ADC input
  3. Quick start — implement ReadChannelV returning volts
  4. Transfer function — understand defaults and when to calibrate
  5. Examples — run the ESP32-C6 project if you use ESP-IDF
