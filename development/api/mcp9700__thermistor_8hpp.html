Detailed Description

Hardware-agnostic MCP9700 / MCP9700A linear active thermistor driver.

Converts the sensor output voltage (0–Vdd) read through your ADC into °C using the factory-calibrated relationship from the Microchip datasheet (typically T_C = (Vout − 0.5 V) / (10 mV/°C)).
