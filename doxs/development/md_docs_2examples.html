ADC ESP-IDF oneshot ADC + optional calibration

Submodule

Initialize shared tooling before ./scripts/build_app.sh:

git submodule update --init --recursive
cd examples/esp32
./scripts/build_app.sh mcp9700_esp32c6_example Debug

Example source

The sketch lives at examples/esp32/main/mcp9700_esp32c6_example.cpp — it wraps hf::mcp9700::Mcp9700Thermistor with an ESP-IDF ADC adapter class.
